# Registry UI

`a2a-warp-registry-ui` is an optional frontend for browsing and managing agents registered with an A2A Warp registry server.

Built with React, TypeScript, and Vite. The UI displays Agent Cards from a registry endpoint and provides basic agent management operations.

The registry UI source lives at `apps/registry-ui/` in the repository and is not published to npm. Use the registry's HTTP API or the `a2a-warp registry` CLI commands for programmatic access.

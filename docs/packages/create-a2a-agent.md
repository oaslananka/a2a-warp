# Scaffolder Package

`create-a2a-warp` scaffolds a new A2A Warp agent project with a working directory structure, package files, and example agent cards.

```bash
pnpm dlx create-a2a-warp my-agent
cd my-agent
pnpm install
pnpm run dev
```

PowerShell:

```powershell
pnpm dlx create-a2a-warp my-agent
Set-Location my-agent
pnpm install
pnpm run dev
```

The scaffolder project resides at `packages/create-a2a-agent/` in the repository.

# a2a-warp health

Checks an A2A endpoint health route.

```bash
a2a-warp health http://127.0.0.1:3000 --timeout-ms 1000 --json
```

```powershell
a2a-warp health http://127.0.0.1:3000 --timeout-ms 1000 --json
```

Shared network options such as `--header`, `--bearer-token`, `--api-key`, `--retries`, `--request-id`, and `--origin` are supported.
